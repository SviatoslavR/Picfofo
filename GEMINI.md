# picfofo - 图床同步上传工具

## 项目概述
picfofo 是一个为 Windows 用户设计的自动化图床工具。
它监听系统截图动作，自动将图片上传至 Cloudflare R2（通过 S3 API），并将生成的 Markdown 链接复制到剪贴板。

## 核心功能
- **动作监听**：监听 PrtScr 或特定热键，或监控指定文件夹（如 Windows 默认截图文件夹）。
- **云端同步**：利用 S3 API 将本地图片同步至 Cloudflare R2 存储桶。
- **链接生成**：自动生成 Markdown 格式链接并写入系统剪贴板。
- **GUI 界面**：提供简单直观的配置界面（配置 R2 凭据、路径等）。

## 技术栈 (初步预设)
- **语言**：Python 3.10+
- **GUI 框架**：PySide6 (Qt for Python)
- **S3 客户端**：boto3
- **热键监听**：keyboard / pynput
- **文件监控**：watchdog

## 开发规范与约束
- **环境隔离**：在安装任何第三方库或软件前，必须询问用户。建议使用 虚拟环境 (`venv` 或 `conda environment`)。
- **凭据安全**：不得将 `access_key` 或 `secret_key` 硬编码或提交至仓库。使用 `.env` 或本地配置文件。
- **代码风格**：遵循 PEP 8，使用类型提示 (Type Hints)。
